# 选择题答题软件 v1.0.0

一款基于 Python + tkinter 开发的 Windows 本地选择题答题软件，支持单选题、多选题和判断题，具有错题记录、收藏、计时等功能。

## 功能特点

- **多种题型支持**：单选题、多选题、判断题
- **Excel导入导出**：支持从Excel导入题目，导出答题记录
- **错题累计**：自动记录每道题的错题次数
- **收藏功能**：可收藏重要题目便于复习
- **计时功能**：每题独立计时，支持暂停/继续
- **多存档管理**：支持多个答题存档，进度自动保存
- **护眼背景**：提供浅绿、米黄、浅灰三种护眼背景色
- **题目导航**：可视化题号网格，颜色区分答题状态
- **快捷键支持**：方向键切换题目，空格显示答案

## 系统要求

- Windows 7/8/10/11
- Python 3.8 或更高版本（运行源码时使用）

## 安装使用

### 方法一：使用打包好的EXE文件

1. 下载并解压软件包
2. 双击运行 `选择题答题软件.exe`
3. 软件会在同目录下自动创建 `saves` 文件夹存储存档

### 方法二：从源码运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行程序
python quiz_app.py
```

### 打包成EXE

```bash
# 运行打包脚本
build.bat
```

打包完成后，可执行文件位于 `dist/选择题答题软件.exe`

## Excel题目格式

导入的Excel文件需按以下格式组织：

| 列 | 内容 | 说明 |
|:---:|:---|:---|
| A | 序号 | 题目编号，从1开始 |
| B | 题型 | single(单选)/multiple(多选)/judge(判断)，可留空自动检测 |
| C | 题干 | 题目内容 |
| D | 选项A | 选项内容 |
| E | 选项B | 选项内容 |
| F | 选项C | 选项内容（判断题可留空） |
| G | 选项D | 选项内容（判断题可留空） |
| H | 选项E | 选项内容（可选） |
| I | 选项F | 选项内容（可选） |
| J | 答案 | 单选填A/B/C/D，多选填ABCD等，判断题A=正确B=错误 |

### 题型说明

- **单选题**：只有一个正确答案，点击选项立即判断
- **多选题**：有多个正确答案，可多选，需要点击确认按钮判断（完全匹配才算对）
- **判断题**：A选项为"正确"，B选项为"错误"

### 答案格式示例

| 题型 | 答案示例 |
|:---|:---|
| 单选题 | `A` |
| 多选题 | `ABD` (多选多个字母，不分先后顺序) |
| 判断题 | `A` (正确) 或 `B` (错误) |

## 操作说明

### 基本操作

1. **导入题目**：文件 → 导入Excel题目
2. **开始答题**：导入后自动加载第一题
3. **答题**：
   - 单选/判断：点击选项立即判断
   - 多选：点击多个选项，然后点击确认
4. **查看结果**：
   - 正确：显示绿色，自动进入下一题
   - 错误：显示红色和正确答案，需手动进入下一题

### 快捷键

| 按键 | 功能 |
|:---|:---|
| `←` | 上一题 |
| `→` | 下一题 |
| `Space` | 显示正确答案 |
| `A`~`F` | 快速选择对应选项 |

### 计时功能

- 每题独立计时，精确到0.1秒
- 点击"暂停"可暂停计时
- 在"视图"菜单中可隐藏计时器

### 错题与收藏

- **错题**：答错的题目会自动累计错题次数
- **收藏**：点击"☆ 收藏"按钮可收藏当前题目
- 错题和收藏题目均可单独导出

### 存档管理

- **新建存档**：文件 → 新建存档
- **加载存档**：文件 → 加载存档
- **另存为**：文件 → 另存为...
- 做题进度自动保存，无需手动保存

### 导出功能

支持导出以下范围的题目：
- 所有已做题目（默认）
- 错题（错题次数>0）
- 收藏题目
- 超时题目（可设置时间阈值）

导出格式：A-L列
- A-J列：原始题目数据
- K列：错题次数
- L列：做题时间（秒，保留1位小数）

### 护眼背景

在"视图"→"护眼背景"中切换：
- **浅绿色** `#C7EDCC` - 默认，最护眼
- **米黄色** `#F5F5DC` - 适合长时间阅读
- **浅灰色** `#E8E8E8` - 简洁风格
- **白色** `#FFFFFF` - 传统白底

### 题目导航

点击"打开导航"或选择"视图"→"题目导航"：
- 显示所有题目的题号网格
- 颜色区分：绿色=答对，红色=答错，金色=当前，灰色=未答
- 点击题号可直接跳转到该题

## 注意事项

1. **首次运行**：软件会在安装目录创建 `saves` 文件夹，请确保有写入权限
2. **数据安全**：存档文件（.db）请定期备份
3. **Excel导入**：请确保Excel文件格式正确，否则可能导入失败
4. **路径问题**：建议使用英文路径，避免中文路径可能导致的问题

## 常见问题

**Q: 导入Excel后没有显示题目？**
A: 请检查Excel格式是否正确，特别是列的顺序和数据类型。

**Q: 多选题如何判断对错？**
A: 多选题需要选择的选项**完全匹配**正确答案才算对，多选或少选都算错。

**Q: 如何修改已答的题目？**
A: 已答题目不支持修改答案，如需重做请新建存档重新导入。

**Q: 计时器的时间如何记录？**
A: 每道题只记录**首次答题**所用的时间，重复查看不会改变时间。

**Q: 存档文件在哪里？**
A: 存档文件（.db）存储在软件目录下的 `saves` 文件夹中。
